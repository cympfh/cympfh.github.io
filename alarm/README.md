# Alarm

シンプルなブラウザアラーム

## 機能概要

- 指定した時刻になると音で知らせる
- 現在時刻とアラームまでの残り時間をリアルタイム表示
- 設定はGUIまたはURLパラメータで行える
- ページを開くだけでアラームがセットされるURLを生成できる

---

## GUI での設定

画面上部の設定パネルで以下を入力し、**Set** ボタンを押す。

| フィールド | 説明 |
|---|---|
| **Time (HH:MM[:SS])** | アラームを鳴らす時刻。24時間制で `07:30` または `07:30:00` のように入力する |
| **Sound** | アラーム音の種類（下記参照） |
| **Auto Stop** | チェックONで、指定秒数が経過すると自動的にアラームが止まる。チェックOFFなら手動でStopボタンを押すまで鳴り続ける |

### Sound の種類

| 値 | 説明 |
|---|---|
| `beep` | 短い電子音 |
| `bell` | 余韻のある鈴音 |
| `chime` | 上昇する和音 |
| `siren` | うなりのあるサイレン |
| `pulse` | 短いパルス音（連打） |

### アラームを止める

アラームが鳴っているときは画面に **Stop** ボタンが現れる。押すと止まる。
Auto Stop が有効な場合は、設定秒数が経過すると自動で止まる（デフォルト: 5秒）。

---

## URL パラメータによる設定

URLにクエリパラメータを付けてページを開くと、**開いた瞬間にアラームがセットされる**。
GUIを操作せずにアラームを仕掛けることができる。

```
https://cympfh.cc/alarm?time=07:30&sound=bell&autoStop=30
```

### パラメータ一覧

| パラメータ | 型 | 説明 | 例 |
|---|---|---|---|
| `time` | `HH:MM` または `HH:MM:SS` | アラーム時刻（24時間制） | `time=07:30` / `time=07:30:00` |
| `sound` | string | アラーム音（`beep` / `bell` / `chime` / `siren` / `pulse`） | `sound=chime` |
| `autoStop` | int | 自動停止までの秒数。**0以下なら自動停止しない** | `autoStop=30` |

### 例

```
# 朝7時30分、ベル音、30秒で自動停止
https://cympfh.cc/alarm?time=07:30&sound=bell&autoStop=30

# 昼12時、チャイム音、自動停止しない
https://cympfh.cc/alarm?time=12:00&sound=chime&autoStop=0

# 深夜0時、サイレン、5秒で自動停止
https://cympfh.cc/alarm?time=00:00&sound=siren&autoStop=5
```

### URLの自動更新

GUIで設定を変更して Set を押すと、ブラウザのURLが自動的に現在の設定を反映したものに更新される。
そのURLをコピーして共有・保存しておけば、次回以降も同じ設定で開ける。

## AI Agent 向け情報

このページは **URLを開くだけでアラームをセットできる** ように設計されている。
AI Agent がユーザーのためにアラームを仕掛ける場合は、以下の手順で対応できる。

1. ユーザーの希望時刻・音・自動停止秒数を確認する
2. 上記パラメータを組み立てたURLを生成する
3. そのURLをユーザーに渡す（またはブラウザで開く）

```
# 例: ユーザーが「明日の朝8時にアラームをかけて」と言ったとき
https://cympfh.cc/alarm?time=08:00&sound=beep&autoStop=30
```

**注意事項:**
- `time` は24時間制 `HH:MM` または `HH:MM:SS` 形式で指定すること
- `autoStop` は整数（秒）で指定。省略時はデフォルト値（5秒）が使われる
- アラームは毎日同じ時刻に繰り返し鳴る（翌日も同じURLで有効）
- 音の再生にはブラウザのユーザー操作が必要な場合がある（autoplay制限）。ページを開いた後、一度クリックやキー操作があれば確実に鳴る

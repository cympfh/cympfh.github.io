陰関数

ここでは二次元上の \[f(x,y) = a\] (\(a\) はある定数) で表現されるグラフのプロットを考える.

関数 \(f\) に簡単な工夫を施すことで \(f(x,y)=0\) とする.

方法

set view 0,0
unset surface
set isosamples 100,100
set contour
set cntrparam levels discrete 0
set xrange [-2:2]
set yrange [-2:2]
splot x*x+x*y+y*y-1

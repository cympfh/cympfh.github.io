% ヒストグラム

`x:(1.0)` からなる折れ線グラフを
`with frequency`
で描くと重複した点の個数だけ `y` を加算する.
これに `with boxes` 等の装飾を施すことでよくあるヒストグラムを描ける.

## 例

```bash
# x-interval
width = 1.0
hist(x) = width * floor(x / width)
set boxwidth width * 0.9
set xtics width

$data <<EOM
1
3
6
8
1
2
2
1
1
EOM

set xrange [0:10]
set yrange [0:]
set ytics 1
set grid
set style fill solid 1.00 border lt -1
plot $data u (hist($1)):(1) smooth frequency with boxes not
```

# 二次元ユークリッド幾何 - 直線, 線分の定義

## 概要

- 直線 `Line` と線分 `LineSegment` の定義
    - 直線はそれが通る二点で定義され, 向きは `(x0, y0)` から `(x1, y1)` に向かうとする
        - `line!(x0, y0; x1, y1)`
    - 線分はそれが持つ二つの端点で定義され, やはり `(x0, y0)` から `(x1, y1)` に向かうとする
        - `line!(x0, y0 => x1, y1)`
- 点までの距離 `::distance_from`
- 同値 `==`, `!=`
    - `1e-6` 程度の誤差を許容
- 線分についてその向きを変える単項マイナス `-`

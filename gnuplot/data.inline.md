# inline data

スクリプトの中で直接データを書いて与える方法.

## 複数行文字列リテラルとして与える方法

`$data` という変数を指定してそれをそのままファイル名の代わりに与える.

@[bash](data.inline.gp)

## 標準入力から与える方法

ファイル名を `'-'` とすると標準入力から読む.
標準入力といってもバッチ実行モードではそのままスクリプトから読む.

`e` のみの行でデータの入力を終わる.

@[bash](data.inline.stdin.gp)

複数 `'-'` をプロットすれば複数回データの入力を読む.
@[bash](data.inline.stdin.multi.gp)

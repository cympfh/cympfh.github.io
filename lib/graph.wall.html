graph.wall.cc

  • 入力;
    • 距離 (コスト) テーブル d
      • d[i][i] = cost: 辺 \((i,j)\) が存在するとき
      • d[i][j] = inf: 辺 \((i,j)\) が存在しないとき
  • 出力 (d の上書き);
    • d[i][j]: パス \(i \rightarrow j\) があるとき、最短距離 (最小コスト)
    • d[i][j] = inf: パス \(i \rightarrow j\) が存在しないとき
void wall(vvi&d) {
  int n = d.size();
  rep (i, n) d[i][i] = 0;
  rep (k, n) rep (i, n) rep (j, n)
    d[i][j] = min(d[i][j], d[i][k] + d[k][j]);
}
